# how to use OSPF simulator
### installation and running simulator

    # git clone https://gitlab.com/abdijavad110/ospf-simulator.git
    # pip install pipenv
    # cd ospf-simulator
    # pipenv sync
    # pipenv shell
    # python main.py

##### add router \<id>

##### add client \<ip>

##### connect \<router1-id/client1-ip> \<router2-id/client2-ip> <bandwidth>
